# LMS-Lenovo-Meta-Search
Privacy Policy

This extension does not collect, store, or transmit personal data.

All data processed by the extension is handled locally within the user's browser. This includes information retrieved from Lenovo support websites and publicly available search engine results for address lookup functionality.

The extension does not use analytics, tracking, or any form of user monitoring. It does not share data with third parties.

Any information copied to the clipboard is done only after user interaction.

This extension is intended solely to provide access to support-related resources and functionality.

If you have any questions, please contact the developer.
